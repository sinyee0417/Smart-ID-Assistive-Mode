Pitch Content

1. Title

Smart ID Assistive Mode: Making Digital Identity Accessible for Every Malaysian
Team: 04 Peeps — Track: Inclusivity

2. Problem Statement

Malaysia is rapidly expanding its digital government ecosystem. However, many citizens still face significant accessibility barriers:

  • Elderly users struggle with small text, complex UI, and hand tremors.
  • Persons with disabilities cannot navigate standard interfaces.
  • Low literacy users have difficulty understanding text.
  • Rural communities have low digital exposure and confidence.

As a result: Exclusion from services, high counter costs, slow digital adoption, and public frustration.

3. Proposed Solution

Smart ID Assistive Mode — a universal accessibility layer triggered when a user taps their Smart ID. The system detects needs and loads an assistive UI.

4–5. How It Works

  1. User taps Smart ID at kiosk/service terminal.
  2. System reads indicators: senior, visual impairment, low literacy, motor disability.
  3. Assistive UI loads automatically.
  4. User navigates via large buttons, voice instructions, icons, or gesture controls.
  5. Service completed smoothly without confusion.

Key Features

  • Large text & buttons
  • Multilingual voice instructions (BM, Mandarin, English)
  • Screen reader mode
  • Icon-based navigation
  • Gesture navigation
  • No-login Smart ID activation
  • Offline-ready kiosks

6. Target Users

  • Senior citizens
  • Elderly / visually impaired
  • Low literacy users
  • Motor disabilities
  • Non-tech-savvy users
  • Rural / low connectivity users

7. Expected Impact

  • Improved digital inclusion
  • Reduced counter queues
  • Faster access to public services
  • Less user frustration
  • Higher Smart ID adoption rate
  • Strengthened Digital Government agenda

8. Feasibility

  • Uses existing Smart ID infrastructure
  • Only requires extra UI layer
  • Low hardware cost
  • Compatible with kiosks & mobile
  • Pilot-ready for rural clinics & district offices

9. Closing

Smart ID should not serve only tech-savvy users. It must be accessible to every Malaysian — regardless of age, ability, or literacy. Smart ID Assistive Mode: Digital Identity for All.
