Smart ID Assistive Mode

Inclusive Digital Identity — Accessible for Every Malaysian

About the Project

Smart ID Assistive Mode aims to ensure that every Malaysian — including seniors, persons with disabilities, low-literacy users, and rural communities — can interact with Smart ID services safely, comfortably, and independently. Our Assistive Mode uses voice guidance, large-button UI, multi-language support, and gesture-based controls to create a seamless, inclusive experience.

UI Mockup Gallery

Key Features

  • Large-text and large-button interface for seniors
  • Icon‑based menu navigation for low-literacy communities
  • Multi-language voice and text support: English, BM, 中文, தமிழ்
  • Built‑in screen reader and voice instruction mode
  • Gesture‑based controls for motor disabilities
  • Instant activation via Smart ID tap — no login needed
  • Offline‑ready for rural digital kiosks

User Flow

1️⃣ Tap Smart ID → 2️⃣ System detects accessibility needs → 3️⃣ Assistive Mode loads → 4️⃣ User follows large‑button or voice‑guided prompts → 5️⃣ Service completed successfully.
