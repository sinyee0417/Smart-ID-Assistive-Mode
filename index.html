Smart ID Assistive Mode

Digital Identity for All — Assistive UI that activates when a citizen taps their Smart ID

What is this demo?

A lightweight prototype demonstrating Assistive Mode: the Smart ID ecosystem automatically switches to a simplified, voice-guided, icon-based UI for seniors, persons with disabilities, low-literacy users, and rural communities.

Quick features

Large buttons

Touch-friendly UI for mobility and vision challenges.

Voice guidance

Multilingual prompts: BM / 中文 / தமிழ் / EN.

Icon navigation

Low-literacy friendly flows using icons and minimal text.

Offline-ready

Works on rural kiosks with limited connectivity.

How it works

  1. Tap Smart ID at a kiosk or terminal.
  2. System reads profile flags and loads Assistive Mode.
  3. User follows large-button, icon-driven flow with voice prompts.
