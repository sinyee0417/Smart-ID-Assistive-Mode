About the Project

项目简介

Smart ID Assistive Mode ensures every Malaysian — seniors, low-literacy users, persons with disabilities, and rural communities — can access digital identity services confidently. This demo showcases our proposed Assistive Mode interface.

智慧身份辅助模式旨在让所有马来西亚人——包括乐龄人士、识字率低的用户、 残障人士和乡区用户——都能够轻松使用数字身份服务。本网站展示我们的辅助模式界面设计。

Key Features

核心功能

  • Large text and buttons for seniors
  • Icon-based menus for low literacy users
  • Multi-language interface: EN, BM, 中文, தமிழ்
  • Screen reader & voice instruction
  • Gesture-based navigation
  • One-tap activation with Smart ID
  • Offline-ready for rural areas
  • 大字、大按钮界面设计
  • 图标式菜单，适合低识字用户
  • 多语言支持：EN、BM、中文、Tamil
  • 语音朗读模式与语音指示
  • 适合行动不便者的手势操作
  • 一键启用，无需登录
  • 支持离线模式，适用于乡区

User Flow

用户流程

1️⃣ Tap Smart ID → 2️⃣ System detects needs → 3️⃣ Assistive UI loads → 4️⃣ User follows guided steps → 5️⃣ Task completed smoothly.
