What this demo shows

This page is a lightweight prototype demonstrating the Assistive Mode concept: auto-adapting UI after tapping a Smart ID at a kiosk.

Key Features

  • Large text & buttons
  • Icon-based navigation
  • Voice instruction placeholders (BM/Chinese/Tamil)
  • Gesture-friendly UI
  • Offline-ready for rural kiosks
See UI Flow

UI Flow (walkthrough)

  1. User taps Smart ID — system detects profile flags (e.g., senior, low vision)
  2. Assistive Mode loads: large buttons + icon menu
  3. Voice guidance starts in preferred language (demo uses text placeholders)
  4. User completes service — printed receipt or confirmation shown
